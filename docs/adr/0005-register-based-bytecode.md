# Register-based bytecode

**Date:** 2026-02-26
**Area:** `bytecode`
**Pull Request:** [#38](https://github.com/frostney/GocciaScript/pull/38)

Register-based bytecode. The VM uses a register machine because it reduces instruction count and avoids redundant operand shuffling compared with a stack VM. [#38](https://github.com/frostney/GocciaScript/pull/38). [bytecode-vm.md § Why Register-Based](../bytecode-vm.md#why-register-based).
